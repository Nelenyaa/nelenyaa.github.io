1. Introduction

Cette politique de confidentialité explique comment je collecte, utilise et protége vos données personnelles lorsque vous utilisez mon site portefolio "Lorenzo Gimenez".

Je collecte différentes informations lorsque vous :

Les informations collectées peuvent être utilisées pour :

  • Améliorer l'expérience utilisateur

4. Cookies

Mon site utilise des cookies pour améliorer votre navigation. Vous pouvez les désactiver dans les paramètres de votre navigateur.

5. Partage des données

Je ne vends ni ne partages vos données personnelles avec des tiers, sauf si cela est nécessaire pour fournir un service ou pour respecter une obligation légale.

6. Vos droits

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données

  • Vous opposer à leur traitement

Pour exercer vos droits, contactez-moi via le formulaire de contact ici.

7. Sécurité

Je mets en place des mesures de sécurité pour protéger vos données, mais aucune transmission sur Internet n'est totalement sécurisée.
